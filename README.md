# Mania-Lab
A set of tools about MUG mania maps.

## Development
- backend: Python 3 (django)
- frontend: undetermined
- dependency: django, ffmpeg

## Done
- convert mc/mcz into osu/osz
- change the speed of maps
- very very simple web interface
